# lambda-crawler

## Requirements

- Node.js v10
- yarn
- serverless

## Install

```
$ yarn install
```

## Usage

Invoke google-search
```
$ yarn invoke:enter
```

Run test

```
$ yarn test
```

Run eslint
```
$ yarn lint
```

Deploy
```
$ yarn deploy [--aws-profile <yourProfile>]
```

Remove
```
$ yarn remove
```
